AutoAPMifier for F5 APM
=======================

Overview
++++++++

AutoAPMifier is an extension for Chrome and Firefox that makes it easier to
access internal resources through an F5 Access Policy Manager Portal.
The extension identifies requests for internal domains, and rewrites the
URLs accordingly.
